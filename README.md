# Apt Path — Student Learning & Career Guidance Platform

## 📌 Overview

Apt Path is a web-based **Student Learning and Career Guidance Platform** designed to help students discover suitable career paths, access recommended learning resources, and follow structured roadmaps for skill development. The platform provides an interactive environment where students can assess their interests, explore career domains, and plan their learning journey effectively.

The system aims to simplify career planning by offering organized learning paths, curated courses, and progress-oriented guidance in a user-friendly interface.

# 🧭 System Workflow

1. User registers and logs into the platform
2. User completes the career assessment
3. System analyzes responses
4. Recommended domains are displayed
5. User accesses recommended courses
6. System generates a career roadmap
7. User follows learning steps and tracks progress

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/your-username/apt-path.git
```

## Navigate to Project Folder

```bash
cd apt-path
```

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```
